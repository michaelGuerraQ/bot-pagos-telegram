🤖 bot-pagos-telegram
Bot de Telegram funcional desarrollado con Node.js para automatizar ventas y enviar alertas de pago/vencimiento en tiempo real.

✨ Funcionalidades

🛒 Venta de productos — flujo automatizado de compra directamente desde Telegram
🔔 Alertas de pago — notificaciones automáticas ante vencimientos sin intervención manual


🛠️ Tecnologías

Node.js
Telegram Bot API


🚀 Instalación
bash# Clonar el repositorio
git clone https://github.com/michaelGuerraQ/bot-pagos-telegram.git

# Entrar a la carpeta
cd bot-pagos-telegram

# Instalar dependencias
npm install

⚙️ Configuración
Crea un archivo .env en la raíz del proyecto con tu token de Telegram:
envTELEGRAM_TOKEN=tu_token_aqui

Obtén tu token creando un bot con @BotFather en Telegram.


▶️ Uso
bashnode index.js
